Algorithm for Experiment 15: Page-Replacement Algorithms

1. Start the program.
2. Read the number of page references (n) and the number of available frames (f).
3. Generate or input a random reference string of length n (values 0-9).
4. FIFO (First-In First-Out) Simulation:
   a. Initialize frames to -1.
   b. For each page in the reference string:
      i. Check if the page is already in one of the frames.
      ii. If not present (page fault):
          - Replace the page in the 'next' frame (circularly incrementing index).
          - Increment the fault count.
5. LRU (Least Recently Used) Simulation:
   a. Initialize frames and a 'last_used' timestamp array.
   b. For each page in the reference string:
      i. Check if the page is in one of the frames.
      ii. If present: Update its timestamp in 'last_used' to the current index.
      iii. If not present (page fault):
          - Find the frame whose timestamp in 'last_used' is the smallest (oldest).
          - Replace that page and update its timestamp.
          - Increment the fault count.
6. Print the fault results for both algorithms.
7. Stop the program.
