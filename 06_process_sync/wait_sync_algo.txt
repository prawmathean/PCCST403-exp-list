1. Start.
2. Call fork() to create a child.
3. In the child process:
   - Print a message and sleep for a few seconds to simulate a task.
   - Exit.
4. In the parent process:
   - Call wait(NULL) which blocks the parent until any child finishes.
   - This ensures the parent doesn't finish before the child.
5. Print final message after wait returns.
6. Stop.
