1. Start.
2. Initialize semaphores: r_mutex, w_mutex, readTry, and resourceAccess.
3. Reader Logic:
   - Wait on readTry (check if writers are waiting).
   - Increment read_count. If first reader, lock resourceAccess.
   - Signal readTry and r_mutex.
   - Read data.
   - Decrement read_count. If last reader, signal resourceAccess.
4. Writer Logic:
   - Increment write_count. If first writer, wait on readTry (blocks new readers).
   - Wait on resourceAccess to get exclusive access.
   - Write data.
   - Signal resourceAccess.
   - Decrement write_count. If last writer, signal readTry (unblocks readers).
5. Join threads and stop.
