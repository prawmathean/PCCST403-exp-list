1. Start.
2. Represent philosophers in states: THINKING, HUNGRY, or EATING.
3. Use a mutex to protect state changes and an array of semaphores (one per philosopher).
4. take_fork(): 
   - Set state to HUNGRY.
   - Check if neighbors (LEFT and RIGHT) are not eating.
   - If safe, set state to EATING and signal own semaphore.
   - Wait on own semaphore to start eating.
5. put_fork():
   - Set state to THINKING.
   - Call test() for LEFT and RIGHT neighbors to see if they can now eat.
6. Each philosopher thread loops through thinking, hungering, and eating.
7. Stop (program runs infinitely).
