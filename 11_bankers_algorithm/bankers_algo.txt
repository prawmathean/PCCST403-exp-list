1. Start.
2. Initialize Allocation, Max, and Available matrices.
3. Calculate the Need matrix: Need[i][j] = Max[i][j] - Allocation[i][j].
4. Maintain a finish array (f[]) initialized to 0.
5. Loop through processes:
   - If process 'i' is not finished and its Need <= Available:
     - Add its Allocation to Available.
     - Mark it as finished (f[i]=1).
     - Add it to the safe sequence.
6. Repeat the loop until all processes are finished or no more processes can be added.
7. If all processes are finished, print the safe sequence. Otherwise, the system is in an unsafe state.
8. Stop.
