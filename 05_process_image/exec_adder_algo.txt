1. Start.
2. Call fork() to create a child process.
3. In the child process:
   - Prepare an array of strings containing the command and its arguments.
   - Call execvp() with the command name and the argument array.
   - If successful, the child process image is replaced and execvp never returns.
4. In the parent process:
   - Call wait() to pause execution until the child finishes.
   - Once the child finishes, print a completion message.
5. Stop.
