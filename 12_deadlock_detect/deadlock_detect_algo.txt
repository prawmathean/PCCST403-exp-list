1. Start.
2. Initialize Allocation, Request, and Available matrices.
3. Set 'Work' equal to 'Available'.
4. For all processes, if Allocation is 0 for all resources, mark as finished. Else, mark as unfinished.
5. Find an unfinished process 'i' where Request[i] <= Work.
6. If found:
   - Work = Work + Allocation[i].
   - Mark process 'i' as finished.
   - Go back to step 5.
7. If any process remains unfinished, a deadlock exists involving those processes.
8. Stop.
